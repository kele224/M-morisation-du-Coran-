Installation sur iOS (iPhone)
Pour installer l'application KELE sur votre iPhone :

1. Ouvrez ce site sur le navigateur Safari.
2. Appuyez sur le bouton de Partage 📤 au bas de l'écran.
3. Faites défiler l'écran vers le bas et sélectionnez « Sur l'écran d'accueil » ➕.
4. Confirmez en appuyant sur Ajouter.
